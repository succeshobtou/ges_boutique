# Gestion de Boutique

## Présentation

Gestion de Boutique est une application web développée avec Laravel pour faciliter l'administration des activités commerciales d'une boutique. Elle offre une solution simple et efficace pour gérer les clients, les produits, les commandes et les factures.

## Fonctionnalités

- Gestion des clients
- Gestion des produits
- Gestion des commandes
- Génération et impression des factures

## Technologies utilisées

- Laravel
- PHP
- MySQL
- HTML5
- CSS3
- Bootstrap

## Installation

1. Cloner le dépôt GitHub.
2. Installer les dépendances :
   ```bash
   composer install
   ```
3. Copier le fichier `.env.example` vers `.env`.
4. Configurer les paramètres de la base de données.
5. Générer la clé de l'application :
   ```bash
   php artisan key:generate
   ```
6. Exécuter les migrations :
   ```bash
   php artisan migrate
   ```
7. Démarrer le serveur :
   ```bash
   php artisan serve
   ```
